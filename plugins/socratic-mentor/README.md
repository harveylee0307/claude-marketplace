# socratic-mentor

蘇格拉底式學習引導 skill。引導使用者自行思考與實作，不直接給完整答案。

## 使用方式

在對話中說出觸發詞即可啟動對應模式：

| 模式         | 觸發詞                                                 | 說明                                  |
| ------------ | ------------------------------------------------------ | ------------------------------------- |
| 引導式開發   | 「引導我」「teach me」「帶我學」「guide me」「帶我做」 | 提供方向與提問，不給完整程式碼        |
| 原始碼閱讀   | 「帶我讀」「source code」「帶我看原始碼」              | 逐段引導閱讀，總結設計思路            |
| Review 清單  | 「review」「幫我看」「檢查一下」                       | 輸出摘要、關鍵決策、學習點、潛在風險  |
| 蘇格拉底問答 | 「考我」「測試我」「quiz me」                          | 依程度出 3-5 題，逐題引導，最後給回顧 |

## 引導原則

- 先問「你目前的想法是什麼？」再回應
- 給提示時說「可以往 X 方向查」而非「你應該用 X」
- 發現觀念有誤時，用反例點出，不直接糾正
- 若使用者明確表示趕時間，可切回一般模式

## 適用情境

- 學習新框架或 API，想深入理解而非複製答案
- 閱讀開源原始碼，需要引導進入複雜 codebase
- 做完功能後想做 code review 並留下學習紀錄
- 驗收學習成果，測試概念理解深度
